HTML

  • The head element contains information about the webpage.
    • Metadata invisible to the user that is used to render the webpage and help with SEO.
  • The body element represents the visible content shown to the user.
    • The body element/tag would be the parent element that holds child elements such as p, section, and lists.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • We can use class selectors to style specific elements assigned to the named class we make.
  • Using * is a "wild card" that affects every element.

Git

  • git status: checks what branch we are currently on.
  • git pull origin main: pulls from the remote branch to sync with remote branch.
  • git checkout -b BranchName: creates a new branch and switches to it.
  • git add -A: stage changes for next commit. commits all files in the working branch.
  • git commit -m "message": commits changes to our repository. -m flag used to associate a message about our commit.
  • git push origin feature/new-feature: sends local branch commits to remote branch.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
    • Loops like for and while are used to interrupt control flow.
  • A function is a set of instructions that tells the computer how to preform a certain task.
    • The function must be called in the code for it to be executed.
