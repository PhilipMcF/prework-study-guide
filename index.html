HTML

  • The head element contains information about the webpage.
    • Metadata invisible to the user that is used to render the webpage and help with SEO.
  • The body element represents the visible content shown to the user.
    • The body element/tag would be the parent element that holds child elements such as p, section, and lists.

Git

  • git status: checks what branch we are currently on.
  • git pull origin main: pulls from the remote branch to sync with remote branch.
  • git checkout -b BranchName: creates a new branch and switches to it.
  • git add -A: stage changes for next commit. commits all files in the working branch.
  • git commit -m "message": commits changes to our repository. -m flag used to associate a message about our commit.
